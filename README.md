# 📏 Размерчик - Telegram Mini App

**Размерчик** - умный помощник для подбора идеального размера одежды и обуви. Хватит гадать, подойдет ли вам «L» от Nike так же, как «L» от Zara.

🚀 **Запустить бота:** [https://t.me/i_know_my_size_bot]

---

## 💡 Личная история (Dogfooding)
Проект вырос из реальной жизни. За 10 месяцев я сбросил **55 кг**, и обновление гардероба превратилось в бесконечный квест с рулеткой. Когда сетки брендов пляшут как хотят, нужен инструмент, который за секунду скажет: «Твой размер здесь - 42.5». 

Я сделал этот MVP за **13 часов**, чтобы закрыть свою боль и помочь другим не ошибаться с заказами онлайн.

---

## 🛠 Технологический стек
Проект реализован в парадигме **Vibe Coding** (фокус на продукте и скорости итераций):

* **Frontend:** React + Tailwind CSS (Hosted on GitHub Pages)
* **Backend:** Node.js + Telegraf (Hosted on Render.com)
* **Database:** Static JSON Data (с легким масштабированием базы брендов)
* **DevOps:** Cron-job.org (хитрый обход «спящего режима» на бесплатном тарифе Render)
* **AI Collaborator:** Gemini (архитектура, отладка логов, рефакторинг)

---

## 🏗 Как это устроено (Under the hood)
Чтобы обеспечить стабильность 24/7 на Zero-budget инфраструктуре:
- **HTTP-мониторинг:** Поднят легковесный сервер для контроля порта и мгновенного прохождения деплоя.
- **Resilience:** Реализован цикл автоматического переподключения к Telegram API для решения конфликтов при обновлении (ошибка 409).
- **Keep-alive:** Настроен внешний «пингер» через Cron-job, который не дает серверу усыпать, поддерживая бота в состоянии постоянной готовности.
